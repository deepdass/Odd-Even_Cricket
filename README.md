# Odd Even Cricket
It is a GUI version of a 2-player hand game that i really love 
## Why 
I made this game because the original game need 2-player for it. In this version the second player is the computer, so you could play the game even you don't have anyone to play with

## How it works
- how the original Odd Even Cricket works
1. At the start, players decide who bats first using an odd/even toss
2. In each round, both players choose a number using their fingers
3. The numbers are added together
4. If the result matches the chosen call (odd or even), the player wins the toss and bats first
5. During gameplay, one player bats while the other bowls
6. Each round both players again show numbers with their fingers
7. If the numbers are the same, the batter is **out**
8. Otherwise, the batter scores runs equal to the number they played
9. After the batter gets out, the players switch roles
10. The second player tries to beat the first player’s score
<br>**The player with the higher score at the end wins the game**

- some important hand signs for runs
1. Showing thumb and little finger Scores - 10 runs
2. Showing only thumb Scores - 6 runs
3. Showing only little finger Scores - 4 runs
4. Showing no fingers Scores - The run equivalent to the other player's sign
